# The Prompt That Built This Game

This game was built entirely by [Claude Code](https://claude.ai/code) from a single prompt:

---

I'm making a Mario-like game that needs to be fully functional — an end-to-end, complete version, not an MVP.

You can use any name you want.

First, design the graphics and images. Choose whatever you prefer: names, libraries, and assets — just make sure the content is not copyrighted. Create a game that is an exact replica of Mario in terms of gameplay structure, with everything set in the foothills of Nepal. The game should include things like Yeti and other elements mentioned in the README.

Before building, do thorough research and plan extremely well what you want to build.

Then start building it. Keep committing continuously, and choose libraries that make the implementation as easy as possible.

The project should be plug-and-play. Someone should be able to run it on their own computer using a simple command, without complicated setup. Make sure this works.

This is an extremely heavy task. I don't want to stop until it's completed.

You cannot ask it after you start running because I'll not be monitoring it. So if you have any questions, ask it now.

---

The result: 3 worlds, 12 levels, 3 boss fights, an original chiptune soundtrack, and ~7,800 lines of hand-written vanilla JavaScript — no engine, no bundler, no dependencies.

See [docs/PLAN.md](docs/PLAN.md) for the full design document Claude produced before writing a single line of code.
